# 스토어 구매 플로우 가이드 (사용자용)

## 📱 전체 구매 프로세스

### 1단계: 상품 탐색

1. **파트너 스토어 방문**
   - 파트너 프로필 페이지에서 스토어 확인
   - 상품 목록 및 카테고리 탐색

2. **상품 상세 확인**
   - 상품 이미지, 가격, 옵션 확인
   - 상품 타입 확인:
     - 📱 **디지털**: 결제 후 즉시 다운로드 가능
     - 🏪 **현장수령**: 결제 후 채팅으로 수령 일정 확정
     - 📦 **택배**: 결제 후 배송 진행
   - 재고 및 배송비 정보 확인

### 2단계: 장바구니에 추가

1. **상품 선택**
   - 옵션 선택 (사이즈, 색상 등)
   - 수량 선택 (디지털 상품은 1개만 가능)

2. **장바구니 추가**
   - "장바구니에 추가" 버튼 클릭
   - ⚠️ **주의**: 같은 파트너의 상품만 장바구니에 담을 수 있습니다
   - 다른 파트너 상품을 담으려면 기존 장바구니를 비워야 합니다

3. **장바구니 확인**
   - 장바구니 아이콘 클릭하여 담은 상품 확인
   - 수량 조정 또는 상품 제거 가능

### 3단계: 주문 및 결제

1. **주문 정보 입력**

   **택배 상품인 경우:**
   - 배송지 정보 입력 (필수)
     - 수령자 이름
     - 연락처
     - 주소 (우편번호 검색)
     - 상세 주소
   - 배송 메모 입력 (선택)
   - 원격지 여부 확인 (추가 배송비 발생 가능)

   **현장수령 상품인 경우:**
   - 수령자 정보 입력 (권장)
     - 이름
     - 연락처
   - 수령 일정은 결제 후 채팅으로 확정됩니다

   **디지털 상품인 경우:**
   - 별도 입력 없음
   - 결제 후 즉시 다운로드 가능

2. **결제 금액 확인**
   - 상품 금액
   - 배송비 (택배 상품인 경우)
   - 총 결제 금액

3. **결제 진행**
   - 포인트 잔액 확인
   - 포인트 부족 시 충전 모달 자동 표시
   - Toss Payments 결제 진행
   - 결제 완료

### 4단계: 구매 후 처리 (상품 타입별)

---

## 📱 디지털 상품 구매 플로우

### 구매 직후

1. **결제 완료**
   - 결제 성공 페이지로 이동
   - 다운로드 권한 자동 부여

2. **다운로드**
   - "다운로드" 버튼 클릭
   - 다운로드 페이지에서 파일 다운로드
   - 구매 목록에서 언제든 다시 다운로드 가능

### 주의사항

- ⚠️ **환불 불가**: 디지털 상품은 결제 후 즉시 다운로드 가능하므로 환불이 불가능합니다
- 다운로드 횟수 제한이 있을 수 있습니다 (상품별 상이)

---

## 🏪 현장수령 상품 구매 플로우

### 구매 직후

1. **결제 완료**
   - 결제 성공 페이지로 이동
   - 스케줄이 생성되지만 시간/장소는 아직 확정되지 않음

2. **채팅으로 수령 일정 확정**
   - 파트너와의 채팅방으로 자동 이동
   - 또는 채팅 목록에서 파트너 선택
   - 원하는 수령 시간과 장소를 파트너에게 알림
   - 파트너가 최종 확정 (예약 완료)

3. **수령**
   - 확정된 시간과 장소에서 상품 수령
   - 파트너가 "수령 완료" 처리

### 특수 상황

**no_show (미수령) 신고:**
- 파트너가 확정한 시간에 수령하지 못한 경우
- "미수령 신고" 버튼 클릭
- 조건 충족 시 자동 환불 처리

### 주의사항

- 수령 일정은 파트너와 채팅으로 조율해야 합니다
- 파트너가 최종 확정 권한을 가집니다
- 근무형 파트너의 경우, 파트너가 출근 중일 때만 수령 완료 처리가 가능합니다

## 📦 택배 상품 구매 플로우

택배 상품은 **협업 제품**과 **개인 제품**으로 나뉩니다.

### 협업 제품 (관리자 출고)

1. **결제 완료**
   - 결제 성공 페이지로 이동
   - 주문 상태: "결제 완료"

2. **출고 대기**
   - 파트너가 관리자에게 출고 요청
   - 관리자 승인 대기

3. **출고 및 배송**
   - 관리자가 출고 승인 및 송장 등록
   - 주문 상태: "배송 중"
   - 주문 상세에서 송장번호 확인 가능

4. **배송 완료**
   - 배송 완료 후 주문 상태: "배송 완료"

5. **구매확정**
   - 상품 수령 확인 후 "구매확정" 버튼 클릭
   - 주문 상태: "구매확정"
   - 정산 및 리뷰 작성 가능

### 개인 제품 (파트너 직접 배송)

1. **결제 완료**
   - 결제 성공 페이지로 이동
   - 주문 상태: "결제 완료"
   - 파트너와의 채팅방에 구매 요청 메시지 자동 발송

2. **배송 준비**
   - 파트너가 상품 포장 및 송장 입력
   - 주문 상태: "배송 중"

3. **배송 정보 확인**
   - 파트너가 채팅으로 택배 정보 발송
     - 택배사
     - 송장번호
     - 배송 조회 링크
   - 주문 상세에서도 배송 정보 확인 가능

4. **배송 완료**
   - 배송 완료 후 주문 상태: "배송 완료"

5. **구매확정**
   - 상품 수령 확인 후 "구매확정" 버튼 클릭
   - 주문 상태: "구매확정"
   - 정산 및 리뷰 작성 가능

### 주문 상태 확인

주문 상세 페이지에서 다음 정보를 확인할 수 있습니다:

- 주문 상태 (결제 완료 → 배송 중 → 배송 완료 → 구매확정)
- 배송 정보 (택배사, 송장번호)
- 배송 추적 링크
- 수령자 정보

## 🛒 장바구니 사용법

### 장바구니에 상품 추가

1. 상품 상세 페이지에서 "장바구니에 추가" 클릭
2. 옵션 및 수량 선택
3. 장바구니에 추가 완료

### 장바구니 관리

- **수량 변경**: +/- 버튼으로 수량 조정
- **상품 제거**: 삭제 버튼으로 상품 제거
- **전체 비우기**: 장바구니 비우기 버튼

### 주의사항

- ⚠️ **파트너 단위 제한**: 같은 파트너의 상품만 장바구니에 담을 수 있습니다
- 다른 파트너 상품을 담으려면 기존 장바구니를 먼저 비워야 합니다
- 디지털 상품은 수량을 1개로 고정할 수 있습니다

## 📋 주문 조회 및 관리

### 내 주문 목록

1. **주문 목록 페이지 접속**

2. **주문 필터링**
   - 전체 주문 보기
   - 상태별 필터링 (결제 완료, 배송 중, 배송 완료, 구매확정)

3. **주문 상세 확인**
   - 주문 카드 클릭하여 상세 정보 확인
   - 주문 상태, 상품 정보, 배송 정보 확인

### 주문 상태별 액션

**결제 완료:**
- 주문 취소 가능 (발송 전)

**배송 중:**
- 배송 추적 (택배 상품)
- 송장번호 확인

**배송 완료:**
- 구매확정 버튼 활성화
- 리뷰 작성 가능

**구매확정:**
- 주문 완료
- 리뷰 작성 가능

## 💰 환불 신청

### 환불 가능 조건

**디지털 상품:**
- ❌ 환불 불가 (결제 후 즉시 다운로드 가능)

**현장수령 상품:**
- ✅ no_show 신고 시 자동 환불
- ✅ 파트너와 협의 후 환불 신청 가능

**택배 상품:**
- ✅ 발송 전: 즉시 환불 가능
- ✅ 발송 후: 반품 배송비 결제 후 환불 가능

### 환불 신청 절차

1. **환불 신청**
   - 주문 상세 페이지에서 "환불 신청" 클릭
   - 환불 사유 입력
   - 환불 신청 제출

2. **환불 처리 대기**
   - 파트너 또는 관리자 검토 대기
   - 주문 상태: "환불 대기"

3. **환불 완료**
   - 승인 시 포인트 자동 환불
   - 거절 시 사유 확인 가능

### 환불 내역 확인

- 환불 목록 페이지에서 환불 내역 확인
- 환불 상태 (대기, 승인, 거절) 확인

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 다른 파트너 상품을 함께 구매할 수 있나요?

**A:** 아니요. 장바구니는 파트너 단위로 제한되어 있어, 같은 파트너의 상품만 함께 구매할 수 있습니다. 다른 파트너 상품을 구매하려면 별도로 주문해야 합니다.

### Q2. 디지털 상품은 환불이 안 되나요?

**A:** 네, 디지털 상품은 결제 후 즉시 다운로드가 가능하므로 환불이 불가능합니다. 구매 전 상품 정보를 꼼꼼히 확인해주세요.

### Q3. 현장수령 상품의 수령 일정은 어떻게 정하나요?

**A:** 결제 완료 후 파트너와의 채팅방에서 수령 일정을 조율합니다. 파트너가 최종 확정 권한을 가지며, 확정된 일정에 맞춰 수령하시면 됩니다.

### Q4. 택배 상품의 배송 정보는 어디서 확인하나요?

**A:** 
- **협업 제품**: 주문 상세 페이지에서 송장번호 확인
- **개인 제품**: 파트너가 채팅으로 배송 정보를 발송하며, 주문 상세 페이지에서도 확인 가능

### Q5. 구매확정은 언제 해야 하나요?

**A:** 상품을 수령하고 확인한 후 "구매확정" 버튼을 클릭하시면 됩니다. 구매확정 후 정산이 진행되며 리뷰를 작성할 수 있습니다.

### Q6. 주문을 취소할 수 있나요?

**A:** 발송 전 주문은 취소 가능합니다. 주문 상세 페이지에서 "주문 취소" 버튼을 클릭하시면 됩니다. 발송 후에는 환불 신청을 해야 합니다.

### Q7. 포인트가 부족하면 어떻게 하나요?

**A:** 결제 시도 시 포인트가 부족하면 자동으로 충전 모달이 표시됩니다. 충전 후 다시 결제를 진행하시면 됩니다.
